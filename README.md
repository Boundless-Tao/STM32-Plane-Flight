# STM32-Plane-Flight
Aircraft Battle Game implemented with STM32F103RC

### How to use

Just download .zip file (plane_fight_CubeIDE.zip/ plane_fight_keil.zip) and unpack it.

There are two version, CubeIDE version and Keil version.
